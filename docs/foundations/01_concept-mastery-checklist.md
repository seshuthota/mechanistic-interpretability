# Concept Mastery Checklist (Pre-Paper)

Mark each item `Yes/No`. Start papers when most are `Yes`.

## Transformer Internals

- [ ] I can explain residual stream in one paragraph.
- [ ] I can explain how one attention head reads and writes.
- [ ] I can explain MLP contribution without saying "black box".
- [ ] I can explain what unembedding does.

## Measurement

- [ ] I can define logit difference and why it is useful.
- [ ] I can name at least 2 behavior metrics and 2 mechanistic metrics.

## Causal Reasoning

- [ ] I can explain the difference between correlational and causal evidence.
- [ ] I can describe what ablation tests.
- [ ] I can describe what activation patching tests.
- [ ] I can describe one failure mode of each method.

## Representation Understanding

- [ ] I can explain superposition/polysemanticity and why SAEs were introduced.
- [ ] I can explain why neuron-level interpretations are often insufficient.

## Research Hygiene

- [ ] I always log model/version/seed/prompt template/metric definition.
- [ ] I can rerun a small experiment and get stable results.

## Ready Signal

You are ready to start papers when:
- at least 12/15 boxes are checked, and
- you can explain all core terms without looking up notes.
